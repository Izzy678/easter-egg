# easteregg

Social media but for movie lovers

## Installation

```bash
npm install
```

## Development

This is a monorepo with frontend (Next.js) and backend (NestJS) applications.

### Start all services:
```bash
npm run dev
```

### Frontend only (apps/web):
```bash
cd apps/web && npm run dev
```

### Backend only (apps/api):
```bash
cd apps/api && npm run dev
```

## Build

To build all applications:
```bash
npm run build
```
